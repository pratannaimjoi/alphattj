Cara Install Bot alphattj Di Termux :
------
- `pkg install nodejs -y`
- `pkg install git -y`
- `git clone https://github.com/tanpattaya/alphattj`
- `pkg install nano`
- `cd alphattj`
- `npm install`
- `npm start`
